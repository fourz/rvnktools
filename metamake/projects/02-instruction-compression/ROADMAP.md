## Compression Strategy

### Content Prioritization

**Essential (Must Keep)**
- Activation phrases and triggers
- Core capability descriptions
- Usage examples
- Integration context

**Important (Optimize)**
- Detailed explanations
- Extended examples
- Supporting context
- Cross-references

**Optional (Consider Removal)**
- Redundant information
- Verbose explanations
- Duplicate examples
- Excessive detail
